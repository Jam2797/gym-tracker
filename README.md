# GYM TRACKER v3 — ПОЛНАЯ ИНСТРУКЦИЯ

## Как обновить на Vercel

1. GitHub.com → твой репозиторий
2. Кликни на index.html → карандаш ✏️
3. Замени весь текст новым
4. Commit changes
5. Vercel деплоит автоматически за ~1 мин

## Что нового в v3

- IKEA-стиль иллюстрации: 2 позиции (до/после) с красными стрелками
- Face pull и пресс для него, боковая планка для неё
- День 3 для неё: кардио + кор + растяжка (не только кардио)
- Деload-неделя каждую 4-ю автоматически (-40% весов)
- Ротация упражнений каждые 18 тренировок
- База 60 продуктов с автоКБЖУ и поиском
- 5 схем питания (Похудение, Поддержание, Набор, Рекомпозиция, Свой)
- Пользовательский ввод ккал
- Расчёт TDEE из профиля по Миффлину
- Календарь питания с цветовыми метками
- Личные рекорды (авто-определение)
- Замеры тела в динамике
- Справочник с поиском по ключевым словам (20+ вопросов)
- Разминочные протоколы для каждого упражнения
- 5 вкладок навигации
