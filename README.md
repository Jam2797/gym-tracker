# 💪 GYM TRACKER PWA

Умный трекер тренировок с адаптивной программой. Работает офлайн на iPhone как нативное приложение.

---

## 🚀 КАК ЗАПУСТИТЬ — ПОЛНАЯ ИНСТРУКЦИЯ

### Шаг 1 — Загрузи файлы на GitHub (бесплатно, 5 минут)

1. Зайди на https://github.com и создай аккаунт если нет
2. Нажми зелёную кнопку **"New"** → создай репозиторий
   - Name: `gym-tracker` 
   - Visibility: **Public**
   - Нажми **Create repository**
3. На странице репозитория нажми **"uploading an existing file"**
4. Перетащи ВСЕ файлы из папки gym-pwa (index.html, manifest.json, sw.js, vercel.json, icon-192.png, icon-512.png)
5. Нажми **Commit changes**

---

### Шаг 2 — Задеплой на Vercel (бесплатно, 2 минуты)

1. Зайди на https://vercel.com
2. Нажми **"Sign Up"** → выбери **"Continue with GitHub"**
3. Нажми **"New Project"**
4. Найди твой репозиторий `gym-tracker` → нажми **Import**
5. Настройки оставь как есть → нажми **Deploy**
6. Подожди ~1 минуту → получишь адрес вида `gym-tracker-xxx.vercel.app`

---

### Шаг 3 — Установи на iPhone (30 секунд)

1. Открой полученный адрес в **Safari** (именно Safari!)
2. Нажми кнопку **Поделиться** (квадрат со стрелкой вверх)
3. Прокрути вниз → выбери **"На экран домой"**
4. Нажми **Добавить**
5. ✅ Готово! Иконка появится на рабочем столе

## 📱 ЧТО УМЕЕТ ПРИЛОЖЕНИЕ

### Для НЕГО (175см / 114кг)
- **3 фазы × ~3 месяца каждая:**
  - Фаза 1: База (техника, лёгкие веса)
  - Фаза 2: Гипертрофия (4×8-12, рост мышц)  
  - Фаза 3: Сила (5×5, тяжёлые базовые)
- **Автоматическая смена фазы** по количеству тренировок
- **Адаптивные веса:** если выполнил все повторения — следующий раз +2.5кг

### Для НЕЁ (168см, травма ключицы)
- **Защита ключицы:** все упражнения с нагрузкой на плечевой сустав сверху исключены или помечены
- **Фокус на ягодицы:** мост, отведения, hip thrust — без риска роста ляжек
- **Домашние тренировки:** Зумба, Йога, Пилатес, HIIT, Плавание — с подсчётом калорий
- **3 фазы:** База → Тонус → Поддержание

### Общее
- **Таймер отдыха** (90 сек) после каждой записи
- **Графики прогресса** по каждому упражнению  
- **Анализ:** частота, прогрессия, рекомендации
- **Дневник питания:** калории, белок, вода с прогресс-барами
- **Работает офлайн** после первого открытия
- **Экспорт данных** в JSON

## ❓ ЧАСТЫЕ ВОПРОСЫ

**Q: Данные сохраняются?**  
A: Да, в localStorage браузера. После переустановки данные сохраняются.

**Q: Работает без интернета?**  
A: Да, после первого открытия. Иллюстрации упражнений требуют интернет.

**Q: Могут ли оба использовать одно приложение?**  
A: Да! Переключатель ОН/ОНА — данные хранятся отдельно.

**Q: Как перенести данные на новый телефон?**  
A: Профиль → Экспорт → сохрани JSON файл.
