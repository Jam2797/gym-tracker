## Что нового в v4

- ИМПОРТ данных из JSON (Профиль → Импорт) — восстановление после сбоя
- Напоминание о бэкапе каждые 14 дней
- Экспорт записывает дату последнего бэкапа
- Калькулятор блинов: какие диски повесить под нужный вес (в упражнении и в Профиле)
- Оценка 1ПМ по формуле Эпли (в упражнении и в Профиле)
- Онбординг при первом запуске: Новичок / Есть база
  - Новичок: старт с -15% весов, Фаза 1 длиннее на 12 сессий
- Заметки к тренировке + настроение (вкладка ПЛАН)
- Выбор дней тренировок (Профиль) + подсказка тренировочный день/отдых на ПЛАНе
- Уровень можно сменить в Профиле в любой момент

## Совет по сохранности данных

На iPhone localStorage может стираться при долгом неиспользовании Safari.
Делай Экспорт раз в 1-2 недели и храни JSON-файл. Импорт вернёт всё назад.
